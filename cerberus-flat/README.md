# 🛢️ Cerberus Tutor 14.5

Tutor interactivo con IA para el Simulador Cerberus 14.5 de NOV CTES.
Powered by Google Gemini AI.

## Estructura
```
cerberus-tutor/
├── api/
│   └── gemini.js        ← Proxy serverless (API Key segura)
├── public/
│   └── index.html       ← App principal
├── vercel.json          ← Configuración Vercel
└── package.json
```

## Deploy en Vercel

1. Crea cuenta en https://vercel.com (gratis)
2. Instala Vercel CLI: `npm i -g vercel`
3. En esta carpeta ejecuta: `vercel`
4. Sigue los pasos en pantalla
5. ¡Listo! Tu URL será: `https://cerberus-tutor-XXXX.vercel.app`
